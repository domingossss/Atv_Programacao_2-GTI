# Atv_Programa-o_2-GTI
Projeto acadêmico desenvolvido para a disciplina de Linguagem de Programação, ministrada pelo professor Leandro Guarino. Consiste em um site de portfólio para um salão de beleza, apresentando serviços, galeria de trabalhos, informações do profissional e canais de contato, aplicando conceitos de desenvolvimento web.
